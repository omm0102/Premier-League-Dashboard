# 英格蘭足球超級聯賽資訊儀表板（Premier League Dashboard）
這是一個我 **獨立開發** 的前端專題作品，整合了英格蘭足球超級聯賽（Premier League）本季的 **賽程、戰績、統計資料、球隊資訊與球員資料**。

本作品完全由前端構成，包含 UI 規劃、資料處理、互動邏輯，並使用 Node.js 做 API 資料前處理。

---

## 功能簡介

### **1. 總覽（Overview）**
- 顯示聯賽亮點（Top 3 球隊、射手榜前三名）
- 顯示聯賽關鍵數字（進行比賽數、總進球數、當前榜首）

### **2. 賽程（Fixtures）**
- 可用「日期」與「球隊」過濾
- 支援重設篩選
- 以卡片方式呈現每天的比賽

### **3. 戰績 / 積分榜（Standings）**
- 顯示 20 支球隊的積分、勝平負與排名
- 依照積分排序，也有 UI 表格呈現

### **4. 統計資料（Stats）**
- 分為三個子面板：
  - 射手榜（Goals）
  - 黃牌榜（Yellow Cards）
  - 紅牌榜（Red Cards）
- 可透過按鈕做切換，前端動態更新內容

### **5. 球隊（Teams）**
- 格狀卡片顯示 20 隊資訊（隊徽、隊名）
- 點擊可開啟 Modal：
  - 球隊教練、主場、成立年
  - 球員名單（節錄）
  - 隊徽、基本資料

### **6. 球員（Players）**
- 支援球員搜尋與球隊篩選
- 以卡片呈現球員照片、姓名、所屬球隊
- 點擊可開 Modal 顯示球員詳細資訊（位置、統計）

### **7. 資料來源（About）**
- 說明 API 來源與英超介紹
- 提供授權與資料使用範圍

---

## 使用技術

### **前端**
- HTML5 / CSS3
- JavaScript（ES6+）
- DOM 操作、事件監聽、互動邏輯
- RWD 響應式設計
- Modal / Tabs / Filter UI Components

### **資料處理**
- Node.js + Fetch API  
  → 用於抓取 APIFootball API 資料並轉存成 JSON  
- 前端再從 JSON 載入資料並渲染 UI

### **外部資料來源**
- APIFootball（賽程、球員、統計）
- Wikipedia / Wikidata（球員照片）
- Premier League 官方網站（比對資料）
